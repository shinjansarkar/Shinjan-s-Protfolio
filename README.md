## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    npm start
    ```

## Customization

- **Personal Info**: Update `components/Hero.tsx`, `components/About.tsx`, and `components/Contact.tsx`.
- **GitHub Stats**: Change the username in `components/GithubStats.tsx`.
- **Resume**: Place your `resume.pdf` in the `public/` folder.
- **Images**: Add your profile picture to `components/Hero.tsx`.
